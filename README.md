# Password Generator

A simple and clean password generator built using HTML, CSS, and JavaScript.

## Features
- Generates random secure passwords
- Displays two passwords at once
- Uses a mix of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols

## How it works
Click the **"Generate Password"** button to instantly create two random passwords.

## Built With
- HTML
- CSS
- JavaScript
